# QuizMod
When you are too busy playing Minecraft to study.

## Example config
Can also be generated by running `/quizmod testconfig` and `/quizmod save`. Note that this will override your existing config.

NOTE: 
```json
{
   "questions":[
      {
         "question":"Did I spend too much time making this mod?",
         "answers":[
            {
               "type":"LiteralAnswer",
               "answers":[
                  "Absolutely",
                  "Yes"
               ],
               "caseSensitive":false
            }
         ],
         "canBeAsked":true
      },
      {
         "question":"What is the capital of Finland?",
         "answers":[
            {
               "type":"LiteralAnswer",
               "answers":[
                  "Helsinki"
               ],
               "caseSensitive":false
            },
            {
               "type":"LiteralAnswer",
               "answers":[
                  "Helsingfors"
               ],
               "caseSensitive":false
            }
         ],
         "canBeAsked":true
      },
      {
         "question":"What determines if your answer to this question is correct?",
         "answers":[
            {
               "type":"RegexAnswer",
               "answers":[
                  "a? (regex|regular expression)"
               ],
               "simple_answer":"a regex or a regular expression"
            }
         ],
         "canBeAsked":true
      }
   ],
   "enabled":true,
   "maxTicksUntilQuestion":20000,
   "minTicksUntilQuestion":1200
}```
